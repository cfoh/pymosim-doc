sim.simsystem module¶

This module provides SimSystem class which is used for simulation system control. SimSystem is a static class, it should be called directly without an instance. It is used for:

  • reporting simulation warning due to incorrect implementation

  • stopping simulation due to a critical error

Note

SimSystem.err.message(err_str)

Call this function when a fatal error has occured. Calling this function will stop the running and report the provided error message.

SimSystem.warn.message(warn_str)

Call this function when a non-critical situation has occured. Calling this function will show a warning on the terminal but will not stop the running.

SimSystem.warn.no_implementation(inspect.stack()[0][3],self)

Call this function to throw a warning in a virtual method. This will not stop the running, a warning will be shown and the program will continue to run. Use this function in a virtual method to warn that a virtual method may not have implemented in a subclass.

static no_implementation(method_name, obj_name)¶

Throw a warning message for no reimplementation of a virtual method in a subclass, but continue to run the program.

This is a static method that should be called directly. This function should be called in a virtual method that expect reimplementation, but may not cause a fatal error if executed. Usage:
